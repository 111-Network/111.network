# Contributing

111 Network is **experimental and in very early development**. Things are changing fast and most decisions aren't settled yet.

If you want to help:

1. Read the [README](README.md) so you know what we're trying to do.
2. Open an [issue](../../issues) or a [discussion](../../discussions) on this repo to say hi and tell us what you'd like to work on.

We'll take it from there.
